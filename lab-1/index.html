Инструкция по использованию

  • Используйте ползунки или поля ввода для изменения значений цветовых компонент
  • При изменении любого параметра в одной модели, значения в других моделях автоматически пересчитываются
  • Используйте цветовые пикеры для выбора цвета из палитры
  • Если значения выходят за допустимые пределы, появится предупреждение о корректировке
  • Текущий цвет отображается в верхней части страницы
