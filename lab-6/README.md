# Отчет по лабораторной работе №6  
**Построение и визуализация трехмерных объектов, реализация трехмерных преобразований и построение проекций**  

---

## 1. Цель работы  
Закрепление теоретического материала и практическое освоение методов трёхмерной визуализации, включая:  
- Построение каркасной модели трёхмерного объекта (буквы "К").  
- Реализацию трёхмерных преобразований: вращение, масштабирование, перенос.  
- Построение трёх ортографических проекций на координатные плоскости.  

---

## 2. Теоретическая часть  

### 2.1 Представление трёхмерных объектов  
Трёхмерный объект задаётся каркасной моделью, состоящей из вершин и рёбер. Вершины описываются координатами в трёхмерном пространстве, а рёбра — парами индексов вершин.

### 2.2 Трёхмерные преобразования  
Преобразования выполняются с помощью матриц 4×4 в однородных координатах:  
- **Вращение** вокруг осей X, Y, Z.  
- **Масштабирование** относительно начала координат.  
- **Перенос** (сдвиг) вдоль осей.  

Матрицы преобразований комбинируются путём умножения, что позволяет применять несколько преобразований последовательно.

### 2.3 Ортографические проекции  
Ортографическая проекция — параллельная проекция на координатные плоскости:  
- **Oxy** — вид сверху (игнорируется координата Z).  
- **Oxz** — вид спереди (игнорируется Y).  
- **Oyz** — вид сбоку (игнорируется X).  

---

## 3. Реализация  

### 3.1 Используемые технологии  
- **HTML/CSS** — структура и стилизация интерфейса в стиле "киберпанк".  
- **JavaScript** — логика преобразований и отрисовки.  
- **Plotly.js** — библиотека для построения 3D- и 2D-графиков.  
- **Math.js** — библиотека для матричных операций.  

### 3.2 Структура приложения  
Приложение состоит из:  
1. **Области 3D-визуализации** (левый блок) — отображает каркасную модель буквы "К" с возможностью интерактивного вращения.  
2. **Панели управления** (правый блок) — содержит элементы управления преобразованиями и кнопки для вызова проекций.  
3. **Модального окна** — отображает три ортографические проекции.  

### 3.3 Алгоритмическая часть  
- **Исходные данные**: массив вершин и рёбер для буквы "К".  
- **Матрица преобразования**: единичная матрица 4×4, обновляется при каждом преобразовании.  
- **Преобразования**:  
  - Вращение вокруг осей X, Y, Z.  
  - Масштабирование.  
  - Перенос.  
- **Проекции**: построение трёх 2D-графиков на основе текущих координат вершин.  

---

## 4. Результаты работы  

### 4.1 Интерфейс  
Приложение имеет современный интерфейс в тёмной цветовой гамме с неоновыми акцентами:  
- Чёткое разделение на зону визуализации и панель управления.  
- Интуитивные элементы ввода для преобразований.  
- Кнопки для вызова проекций и сброса преобразований.  

### 4.2 Визуализация  
- **3D-модель**: линии — неоновый циан, вершины — яркая маджента.  
- **Координатные оси**: подписаны, сетка отображается для удобства ориентации.  
- **Проекции**: три отдельных графика в модальном окне с подписями осей.  

### 4.3 Функциональность  
- Корректное применение последовательных преобразований.  
- Реальное время обновления графика при изменении параметров.  
- Вывод итоговой матрицы преобразования в текстовом поле.  
- Возможность сброса к исходному состоянию.
